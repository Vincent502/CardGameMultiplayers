Dossier des rapports de parties (historique).
Les fichiers .log sont générés automatiquement à chaque partie terminée.
Format : cardgame_AAAAMMJJ_HHmmss.log

Emplacement des rapports générés (build Windows/Android) :
  persistentDataPath/Rapport/
  - Windows : %USERPROFILE%\AppData\LocalLow\<Company>\<Product>\Rapport\
  - Android : /data/data/<package>/files/Rapport/
