# Jeu de carte

- 34 Cartes (4 obligatoire)
- carte a différent types

### Types de carte

- **Equipé** : carte place sur le board et effective jusque la fin de partie est automatiquement posé sur le board
- **Normal** : carte placé dans le cimetiaire apres utilisition
- **Ephémaire** : Carte utilisable une fois par partie
- **Rapide** : carte jouable uniquement pendant le tour adverse

### Systeme de tour

- **Tour 1** : 3 cartes 1 point de mana
- **Tour 2** : 4 cartes 2 point de mana
- **Tour 3** : 5 cartes 3 point de mana
- Appartire du troisieme tour il n'y a plus de carte et de mana supplementaire

### Déroulement du tour

- les cartes tiré précédemant sont jeté
- Lorsque le tour commence les bouclier sont remis a 0
- Résolution des effets de cartes et équipement
- tirage des cartes, si le deck vide reprendre le cimetiaire, le mélanger et le replacer au deck pour prendre des cartes, si le deck possède encore des cartes mais pas 5 il ne jouera qu'avec les cartes restante.
- le joueur joue
- résolution des effets de cartes et equipements
- fin du tour

```
1 tour = 1 joueur
```

### Valeur du joueurs

- **Force** : Augmente les dégats
- **Resitance** : Augment le bouclier
- **Bouclier** : Valeur d'armure pour encaisser les degats
- **PV** : Point de vie du joueur (100PV)
- **Mana** : Energie a dépense pour joué des cartes

### Condition de victoire

  La victoire est accordée au joueur ayant reduis les point de vie adverse a Zero
